# [ATS3](http://www.ats-lang.org/)/srcgen2/xats2cz

This directory contains versions of XATS2CZ (ATS3-to-Scheme compiler)
written in ATS3. Note that 'CZ' in the name 'XATS2CZ' refers to Chez
(as in the Chez compiler for Scheme).

## The [xats2py/srcgen1] subdirectory

This is the place where one finds the first version of XATS2CZ written
in ATS3, which can be referred to as xats2cz/srcgen1.

Yes, xats2cz/srcgen1 can be used to bootstrap ATS3. While this
compiler does not do tail-recursion optimization (TRO) , it is
expected that a Scheme compiler (like Chez) performs TRO on the code
generated by xats2cz/srcgen1. Without TRO, the runtime stack needs to
be set to a considerably large size (e.g., 10K) in order to support
boostrapping.
